# LightRender

## Descripción

LightRender es una aplicación web desarrollada en Angular que permite la simulación visual de iluminación en un entorno interactivo. El proyecto está enfocado en la construcción de una landing page modular como base para futuras funcionalidades del simulador.

---

## Nota sobre archivos del proyecto

Dentro del repositorio existen algunos archivos y carpetas como:

* `node_modules/`
* `package.json`
* `package-lock.json`
* `server.js`
* `.DS_Store`

Estos archivos **no forman parte del primer sprint funcional del proyecto**.

Fueron incluidos únicamente con fines de prueba para:

* Verificar la configuración del entorno de desarrollo
* Asegurar que el proyecto pudiera ejecutarse correctamente
* Preparar una posible integración futura con el backend del simulador

---

## Estado actual

En este primer sprint, el enfoque principal fue:

* Desarrollo de la estructura de la landing page
* Implementación de componentes en Angular (Navbar, Hero, Technologies, Audience, Business, Purpose, Future, CTA y Footer)
* Organización modular del frontend
