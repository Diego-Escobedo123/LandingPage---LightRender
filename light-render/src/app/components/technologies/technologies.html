Stack tecnológico

Construido con tecnologías web modernas

LightRender utiliza tecnologías de renderización gráfica en tiempo real que corren directamente en el navegador, sin instalaciones ni plugins externos.

WebGL + Three.js

Motor de renderizado 3D acelerado por hardware. Permite simular iluminación, materiales, shading y efectos visuales en tiempo real desde el navegador.

Angular (Frontend)

Framework para construir la interfaz del usuario, los controles interactivos y la integración del entorno 3D con la lógica de la aplicación.

Node.js + Express

Backend con API REST para gestionar productos, usuarios y configuraciones de simulación. Sirve como puente entre el frontend y la base de datos.

MongoDB

Base de datos NoSQL para almacenar información del sistema, configuraciones de productos y datos de usuarios de forma flexible y escalable.

Blender

Herramienta auxiliar para modelar la geometría 3D de la bombilla con alto nivel de detalle: filamento, bulbo y componentes internos.

HTML5 + CSS3

Base estructural de la aplicación. Permiten integrar el entorno 3D con controles visuales y garantizan compatibilidad en cualquier navegador moderno.
